# Personal Dashboards

This repository contains a proposal for a nonfiction book about building simple personal analytics dashboards with commercial tracking hardware/software (e.g. activity trackers, wifi-enabled scales and body composition analyzers, etc.) IFTTT, and Google Data Studio.
